# 실시간 한국어 → 베트남어 자막 시스템

예배 중 한국어로 말하면 베트남어 자막이 실시간으로 표시됩니다.
저사양 PC에서도 동작합니다 (모든 처리는 클라우드에서 수행).

---

## 방법 1: 웹 버전 (가장 쉬움, 설치 불필요)

### 필요한 것
- Chrome 브라우저
- 마이크
- 인터넷 연결

### 사용법
1. `web/index.html` 파일을 Chrome으로 엽니다
2. **시작** 버튼을 클릭합니다
3. 마이크 접근을 허용합니다
4. 한국어로 말하면 베트남어 자막이 표시됩니다

### 예배 시 화면 구성

#### 방법 A: 화면 분할 (가장 간단)
```
┌─────────────────────────┐
│                         │
│     PowerPoint 슬라이드   │
│      (예배 가사)          │
│                         │
├─────────────────────────┤
│   베트남어 자막 (Chrome)   │
└─────────────────────────┘
```
1. PowerPoint를 창 모드로 실행 (화면 위쪽 75%)
2. Chrome을 화면 아래쪽 25%에 배치
3. Chrome에서 **프로젝터 모드** 버튼 클릭

#### 방법 B: OBS 사용 (더 깔끔)
1. OBS Studio 설치 (무료)
2. Scene에 PowerPoint 화면 캡처 추가
3. Scene에 Chrome 브라우저 창 캡처를 하단에 추가
4. OBS 가상 카메라 또는 전체 화면 출력

## 방법 2: Python 오버레이 (PPT 위에 자막)

PowerPoint 위에 투명한 자막 창이 떠서 가장 깔끔합니다.

```
┌─────────────────────────┐
│                         │
│     PowerPoint 슬라이드   │
│      (전체 화면)          │
│                         │
│  ┌───────────────────┐  │
│  │ 베트남어 자막 (투명) │  │
│  └───────────────────┘  │
└─────────────────────────┘
```

### 필요한 것
- Python 3.8 이상
- 마이크
- 인터넷 연결

### 설치 (Windows)
```
cd python
install_windows.bat
```

또는 수동 설치:
```
pip install SpeechRecognition deep-translator PyAudio
```

### 실행
```
cd python
python app.py
```

### 조작법
- **드래그**: 자막 창 위치 이동
- **ESC**: 종료
- **✕ 버튼**: 종료

## FAQ

**Q: 인터넷이 꼭 필요한가요?**
A: 네, 음성인식과 번역 모두 클라우드 API를 사용합니다. 대신 저사양 PC에서도 잘 동작합니다.

**Q: 번역 품질이 어떤가요?**
A: Google 번역 수준입니다. 완벽하지는 않지만 의미 전달은 충분합니다.

**Q: 무료인가요?**
A: 네, 모두 무료 API를 사용합니다. 웹 버전의 MyMemory API는 하루 5,000단어 제한이 있으나, Google 비공식 API로 전환하면 제한 없이 사용 가능합니다.

**Q: 지연 시간은 얼마나 되나요?**
A: 보통 1~3초 정도입니다 (문장 끝난 후 번역까지).

**Q: PyAudio 설치가 안 돼요**
A: Windows에서 아래 명령어를 시도해보세요:
```
pip install pipwin
pipwin install pyaudio
```
